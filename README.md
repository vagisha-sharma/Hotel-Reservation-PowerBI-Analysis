## Project Overview
This project analyzes a hotel reservation dataset using Power BI to extract insights related to revenue, customer behavior, operational efficiency, cancellations, seasonal trends, and predictive forecasting.

The dashboard consists of 8 analytical pages covering different business domains.

---

## Dashboard Pages

1. Sales & Revenue Performance Overview  
2. Customer & Booking Behavior Analysis 
3. Operational Efficiency & Occupancy Analysis  
4. Cancellation Behavior & Revenue Impact Analysis  
5. Time-Based Booking & Revenue Trends  
6. Customer Demographics & Segmentation Analysis  

---

## Key KPIs Analyzed

- Total Revenue  
- Average Daily Rate (ADR)  
- Revenue per Available Room (RevPAR)  
- Cancellation Rate  
- Average Lead Time  
- Occupancy Trends  
- Repeat Guest Rate  

---

## Tools Used

- Power BI  
- Power Query  
- Basic DAX Measures  
- Forecasting (Analytics Pane)

---

## Dataset Features

- Booking details (adults, children, nights)
- Pricing information
- Room type
- Market segment
- Booking status
- Lead time
- Arrival date details

---

## Key Insights

- Online bookings contribute majority of revenue.
- Longer lead times show higher cancellation probability.
- Seasonal booking peaks observed mid-year.
- Adults dominate overall guest composition.
- Room Type 1 shows highest utilization.

---

## How to Use

1. Download the `.pbix` file.
2. Open using Microsoft Power BI Desktop.
3. Use slicers to filter by year and explore insights.
